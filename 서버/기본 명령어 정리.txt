pwd
현재 있는 디렉토리 확인
예) /home/practice/test

history
지금까지 쳤던 명령어 보여주는 것

sudo
일반 사용자가 루트 명령어를 사용할 수 있음
예) sudo passwd practice

useradd
새로운 사용자를 생성하는 명령어
예) useradd practice

passwd
사용자의 패스워드를 바꾸는 것
예) passwd practice 

chmod
사용자가 읽고 쓸 수 있는 권한을 주는 것
예) chmod 777 ju 

chown
파일의 소유자를 바꿔주는 것
예)chown practice:practice ju

cd
경로 접근
예) cd /home/practice

./
현재 경로
../
전 경로

ls-al
안에 있는 파일 디렉토리 전부

cp
파일 복사
예) cp ju ../kkk

mv
이동
예) mv kkk /home/practice/test/kkkk

touch
파일 만드는것
예) touch pyo

vi
파일 만드면서 바로 내용도 첨부 가능 (편집기)

cat
파일 내용 확인
예) cat pyo

mkdir
디렉토리 생성
예) mkdir chu

rm(파일 삭제)
에) rm kkkk
rm -r(디렉토리 삭제)
rm -r chu
rm -f(그냥 삭제)

하드링크

예) ln pyo ../

소프트링크
포인터처럼 원본을 가리키는 것 원본이 사라지면 사용 불가능
예) ln -s /home/practice/test/hw ../iiiiii
ln -s [링크파일] [생성할파일]
